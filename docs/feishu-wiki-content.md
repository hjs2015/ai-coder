## 💡 核心知识点总结

### AI 编程工具三大分类

1. **Chat 聊天型**
   - 特点：自然语言对话，零门槛
   - 代表：ChatGPT、Claude、Gemini
   - 场景：代码咨询、片段生成、Bug 排查

2. **IDE 集成型**
   - 特点：原生 AI IDE，沉浸式编码
   - 代表：Kiro、Cursor、Antigravity
   - 场景：专业项目开发、日常编码

3. **CLI 命令行型**
   - 特点：纯终端交互，高效批量处理
   - 代表：Codex、Claude Code
   - 场景：服务器开发、自动化部署

### 主流 AI 模型厂商

**国际厂商**：
- OpenAI：GPT-5.4、GPT-5.3-Codex
- Anthropic：Claude Opus 4.6、Claude Sonnet 4.6
- Google：Gemini 3.1 Pro

- 阿里：通义千问 Qwen 2.5 Code
